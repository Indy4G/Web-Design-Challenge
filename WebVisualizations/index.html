Summary: Latitude Vs. X


How does weather change based on one's distance from the equator? This study endeavors to answer this question. Using weather data pulled from the OpenWeatherMap API, visuals were developed in order to illustrate changs in four different weather patterns in over 500 cities based on each city's latitudinal location relative to the equator. These four weather patterns include: Max Temperature, Humidity, Cloudiness, and Wind Speed. For more details on findings select a visual or see the "Plots" drop-down menu on the navigation bar.
